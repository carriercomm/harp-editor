# Harp Editor PREVIEW - PRE-ALPHA 

Adds ACL and other niceities to Harp Boilerplates.
Built on top of Harp web server and uses best practices.

Note the version is 0.0.0

	$ npm install
	$ node server 

## Features
- Users
- Admin Dashboard
- Decorates JSON
- Saves Locally
- Deployes to Harp Platform [Not done]
