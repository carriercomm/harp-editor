# Harp Editor PREVIEW

Adds Editor and CMS like Admin interface on top of any Harp Boilerplates.
Built on top of Harp web server and uses best practices.

Note the version is 0.0.1

	$ npm install
	$ node server 

## Features
- Users
- Admin Dashboard
- Decorates JSON
- Saves Locally
- Deployes to Harp Platform [Not done]
